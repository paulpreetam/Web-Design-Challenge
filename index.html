Summary: Latitude vs. X

The purpose of this project was to anaylze how weather changes as it gets closer to the equator. To accomplish this anaylsis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, windspeed and humidity. This site provides source data and visualizations created apart of the anaylsis, as well as an explanations and descriptions of any trends and correlations witnessed.
